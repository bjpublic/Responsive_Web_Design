이번 장에서 사용하는 reset.css에는 이 책에서 소개하는 [html5doctor Reset Stylesheet]을
베이스로 하여 책 내의 CSS 작성을 간략화 하기 위하여 모든 reset.css에 대해 
아래의 코드를 추가하고 있습니다.

-------------------------(이번 장의reset.css에 추가한 코드)---------------------------

/* 이 책 샘플용으로 아래의 코드를 추가하고 있습니다. */
ol, ul {
    list-style: none;
}
/* clearfix */
.cf {
	zoom: 1;
}
.cf:before, .cf:after {
	content: "";
	display: table;
}
.cf:after {
	clear: both;
}

-------------------------------------------------------------------------------------

이 코드들은 reset.css가 아닌 각 샘플 CSS를 추가하는 것도 가능합니다만
모든 페이지에서 공통 스타일이 되므로 본 샘플에서는 reset.css를 추가하고 있습니다.
이 책을 읽어나간 후에 이점에 주의하여 주십시오.
